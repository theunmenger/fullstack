Een project voor het ROC van flevoland.
De opdrach was om een webapplicatie te ontwikkelen om het beheer van voorraad te automatiseren voor een fictief bedrijf genaamd "ToolsForEver".

Gebruikte talen:
  -js
  -css
  -html
  -php

Structuur:
index is een login pagina
dashboard.php is je landingspagina, hieruit kan je je producten kan inzien en door kan klikken naar bestellen of bewerken.
In je header kan je naar dashboard.php, add_product.php, add_fabriek.php, add_locatie.php, bestellingen.php, settings.php, admin.php (alleen toegankelijk voor admin accounts) en je kan uitloggen.
In bewerken.php wordt je doorgestuurt naar diverse php pagina's afhankelijk van de optie die je hebt aangeklikt.
In bestellen.php kan je producten bestellen.
In bestellingen.php kan je je bestellingen inzien.
In settings.php kan je worder doorgestuurt naar diverse (optie)_change.php pagina's afhankelijk van de optie die je hebt aangeklikt.
In admin.php kan je op add_account.php klikken

Verbetervoorstel:
Ik zou meer comments willen toevoegen en minder php willen mixen in mijn html.
Ook zou ik op een beter tempo gewerkt kunnen hebben.
